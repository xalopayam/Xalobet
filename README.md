# XaloBet - معامله کریپتو و شرط‌بندی ورزشی

![XaloBet Logo](https://via.placeholder.com/150x50/000000/FFD700?text=XaloBet) <!-- بعداً لوگو واقعی اضافه کن -->

به **XaloBet** خوش آمدید! سایت حرفه‌ای برای خرید/فروش تتر و بیت‌کوین، پیش‌بینی ورزشی و کازینو آنلاین با طراحی لوکس (مشکی-طلایی).

## ویژگی‌های اصلی
- نرخ زنده دلار، تتر (USDT) و بیت‌کوین (BTC) از API واقعی (CoinGecko)
- بخش معامله کریپتو با فرم سفارش
- پیش‌بینی ورزشی با ضرایب نمونه (قابل ارتقا به API واقعی)
- کازینو زنده با بازی انفجار (canvas انیمیشن)
- کیف پول داخلی (ریال + کریپتو) با شارژ شبیه‌سازی
- ثبت‌نام/ورود با شماره موبایل
- کاملاً ریسپانسیو (موبایل/دسکتاپ) + RTL فارسی
- امنیت JWT و bcrypt

## تکنولوژی‌ها
- Backend: Node.js + Express
- Frontend: HTML5, CSS3, JavaScript vanilla
- Dependencies: express, axios, jsonwebtoken, bcryptjs, cors
